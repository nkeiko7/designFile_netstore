タコピザ

●材料（2人分）

  • 炒めごはんの素タコライス 1袋
  • 強力粉 200g
  • オリーブオイル 大さじ1
  • 塩・砂糖 少々
  • ぬるま湯 100～150cc
  • うち粉 適量
  • たまねぎ 1/2個（スライスする）
  • エリンギ 大1本（スライスする）
  • とろけるチーズ 4枚（適当な大きさにきる。ピザ用チーズでもOK）
  • パセリ 適量

●つくり方

  1. 強力粉・塩・砂糖を合わせたボウルにオリーブオイル、ぬるま湯を加え、混ぜ合わせてこねる。（ぬるま湯は生地の固さを見ながら加えていく）
  2. よくこねて生地はひとつにまとまり、表面がなめらかになったらラップに包んで5～10分休ませる。
  3. 休ませた生地を、打ち粉をしいたまな板に乗せ、直径20cm程度の円に伸ばす。（半分に分けて、2枚の生地にする）
  4. フォークで生地全体に穴をあけ、炒めごはんの素タコライスをたっぷりぬり、その上にスライスしたたまねぎ、エリンギを乗せ、とろけるチーズをまんべんなくかける。
  5. 220℃に予熱したオーブンで15分焼き、最後にパセリをふりかけて完成。

沖縄風エビチリ

●材料（2人分）

  • 炒めごはんの素タコライス 1袋
  • むきえび 10尾
  • チンゲンサイ 1株
  • 卵 3個
  • 塩こしょう 少々
  • ごま油 適量

●つくり方

  1. 卵に塩こしょうを加え混ぜる。フライパンにごま油を熱し、半熟状のいり卵をつくって、皿へ盛る。
  2. 1のフライパンに再びごま油を熱し、えびとチンゲンサイを炒める。火が通ったら炒めごはんの素タコライスを加えてからめる。
  3. 1のいり卵の上に、2をのせてできあがり。

ジャンバラヤのパングラタン

●材料（2人分）

  • 炒めごはんの素ジャンバラヤ 1袋
  • 卵 2個
  • 生クリーム 80cc
  • 食パン 2枚
  • ピザ用チーズ 適量

●つくり方

  1. ボールに炒めごはんの素ジャンバラヤ、卵、生クリームを入れ、よく混ぜる。
  2. 食パンはひと口大にきり、グラタン皿に入れる。
  3. 2に1をかけて、ピザ用チーズをのせ、180度のオーブンで約10分焼く。

ケイジャン風コテージパイ

●材料（4人分）

＜マッシュポテト＞
じゃがいも 大2個（400g）
（A）
牛乳 大さじ3程度
バター 大さじ1
塩、こしょう 適量
＜フィリング＞
炒めごはんの素ジャンバラヤ 1袋
牛ひき肉 200g
玉ねぎ（みじん切り） 小1個
油 適量
＜つや出し＞
卵黄 適量

●つくり方

  1. じゃがいもは皮をむいて水から茹で、火が通ったらつぶして（A）をまぜてマッシュポテトにする。
  2. フライパンに油を熱し、牛ひき肉を炒め、玉ねぎを加えてさらに炒める。
  3. 2に炒めごはんの素ジャンバラヤを加えて調味する。
  4. 耐熱容器に3のフィリングを敷き詰め、1のマッシュポテトを重ねて盛りつける。
  5. 好みで表面にフォークで模様をつけ、水で薄めた卵黄を表面に塗る。
  6. 200度のオーブン（オーブントースターでも可）で焼き、表面に焦げ目がついたらできあがり。お好みでパセリなどを添えてどうぞ。

ジャンバラヤ じゃがいも もち

●材料（2～3人分）

  • じゃがいも 250g（中3個）
  • 片栗粉 1／4カップ
  • 海苔 適量
  • 炒めごはんの素ジャンバラヤ 1／2袋

●つくり方

  1. じゃがいもの皮を剥いてゆでる。粉ふかすように鍋で乾煎りし、水分を飛ばし、マッシャーなどで丁寧につぶす。
  2. つぶしたじゃがいもに、片栗粉と炒めごはんの素ジャンバラヤを混ぜる。
  3. ラップで棒状に形を整えて包み、冷蔵庫で1時間程度冷やす。
  4. 冷やしたものを1cmくらいの厚さに切る。フライパンにバターを入れて、両面をこんがり焼き、海苔を巻いてできあがり！

しょうゆ 大さじ1.5

ごま油 大さじ0.5
おろしにんにく 1／2かけ

●つくり方

  1. 小松菜は3cm幅くらいに切り、万能ねぎは5mm幅くらいに刻んでおく。
  2. フライパンを熱してサラダ油大さじ1をひき、とりひき肉を入れ、とりひき肉に火が通ったらごはんを入れる。ごはんをほぐし、炒めごはんの素ナシゴレン半袋を加えて炒める。よく混ざったら万能ねぎ入れ、軽く炒め、火を止める。
  3. 2をティースプーン1杯くらいを餃子の皮にのせて包む。
  4. 大きめの鍋に水と（A）を加え、中火にかけて沸騰したら3を入れる。再び沸騰したら弱火で2、3分茹で、小松菜を入れたら1分くらいで火を止める。

ジューシー豆腐ハンバーグ

●材料（3人前）

  • 木綿豆腐 350g
  • 片栗粉 大さじ2
  • 炒めごはんの素 ジューシー（沖縄風ごはん） 1袋

●つくり方

  1. 木綿豆腐をレンジで30秒ほど過熱し、水分をきる。
  2. 1をボールに入れ、ジューシー（沖縄風ごはん）の素1袋とよく混ぜあわせる。
  3. 2に片栗粉を入れよく混ぜ、適当な大きさに成型する。
  4. フライパンで成型したハンバーグを両面焼いてできあがり。
